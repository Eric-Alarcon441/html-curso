Formatos de texto HTML

Etiqueta b, sirve para poner texto ennegrita

Etiqueta strong, sirve para poner texto en negrita pero en semantica nos indica que este texto es importante

Etiqueta i, sirve para poner el texto en italica

Etiqueta em, sirve para poner el texto en italica pero en semantica indica que esta enfatizado el texto

Etiqueta small, sirve para poner el texto mas pequeño

Etiqueta mark, sirve para resaltar el texto

Etiqueta del, sirve para tachar el texto

Etiqueta ins, sirve para subrayar el texto e indica que el texto fue insertado

Etiqueta sub, sirve para Poner un sub índice en el texto, ejemplo H2O

Etiqueta sup, sirve para poner un Super índice en el texto

Referencias de caracteres en HTML

Etiqueta code, sirve para mostrar el texto como código

Chart ref ingresando a la página web https://dev.w3.org/html5/html-author/charref podemos ver y usar los valores de los caracteres para incluirlos en nuestra página: ¼ ⅛ ω ‱ @ ingresarlos de esta forma nos ayuda a que siempre se visualicen de forma correcta nuestros caracteres.

Podemos utilizarlo de 3 diferentes maneras:

! Símbolo de exclamación

! Símbolo de exclamación hexadecimal

! Símbolo de exclamación decimal

Sirve para utilizar caracteres restringidos como < > ! y que no afecten en la sintaxis del código y de igual forma con carácteres que den conflicto.
