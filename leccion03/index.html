Introducción a CSS

Dentro del head podemos hacer uso de la etiqueta style que nos permite ingresar código CSS, lo recomendable es crear una carpeta y crear un archivo css e insertar ahí el código, debemos agrear una etiqueta link en el head para enlazar los archivos

La ventaja de tener el código css por separado es que podemos aplicar los mismos stilos a diferentes archivos html

Ir al contenido

Se llaman hojas de estilo en cascada por que se aplican desde los elementos padre hasta los hijos

cascadeo en las hojas de estilo.

Los elementos padre aplican sus atributos a los hijos, para que un hijo sea diferente debe tener sus propios atributos

La jerarquia que se maneja es la siguiente

  • 1.- Estilo en etiqueta html
  • 2.- Estilo dentro de la etiqueta style
  • 3.- Estilo en archivo externo

Box model

Tenemos por defecto en los elementos de html lo siguiente

Padding, border, margin

Todas las propiedades tienen sus varciones para cada uno de sus lados

margin-top
margin-right
margin-bottom
margin-left

Podemos obviar el ingresar cada uno y usar la propiedad margin, padding o border con 4 valores que tiene el orden:

top-right-bottom-left
si a la propiedad se le escriben 3 valores
margin: top left-right bottom
si a la propiedad se le escriben 2 valores
margin: top-bottom left-right
si a la propiedad se le escribe 1 valor
margin: top-bottom-left-right
si a la propiedad se le escribe el valor auto, centra de manera horizontal
margin: auto

Para centrar el texto utilizamos la propiedad text-align: center
